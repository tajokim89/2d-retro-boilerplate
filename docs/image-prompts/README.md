# 이미지 프롬프트 가이드

`docs/image-prompts/` 안의 MD 파일들은 **AI 에이전트(Claude CoWork, ChatGPT, SD)에게 픽셀 아트를 시킬 때** 그대로 복사해서 쓸 수 있는 프롬프트 템플릿입니다.

## 워크플로우

```
1. 프롬프트 템플릿 선택  →  2. 이미지 생성 (ChatGPT/SD 등)
                            ↓
3. Aseprite 로 슬라이스/정리 (Claude CoWork 위임 가능)
                            ↓
4. spritesheet + JSON export → public/assets/sprites/
```

## 공통 스타일 가이드

이 보일러플레이트의 기본 가정 — **호러 탐험 톤**. 포크 후 변경 가능하나 한 작품 안에서는 통일성 유지.

| 항목 | 기본값 |
|---|---|
| 타일 사이즈 | 16 × 16 px |
| 캐릭터 사이즈 | 16 × 24 (머리 위로 살짝 길게) |
| 시점 | **탑뷰(top-down) 또는 약간의 3/4 view** — 한 작품 안에서 통일 |
| 팔레트 크기 | 12~16색 (차가운 톤 베이스 + 한두 개의 절제된 액센트) |
| 외곽선 | 픽셀 단색 외곽선, anti-aliasing 없음 |
| 그림자 | 단색 hard drop shadow (캐릭터/추적자) |
| 톤 | **차분하고 낡고 차가움.** 화려한 RPG 색감 X. 형광 색 X. |

## 파일별 가이드

- [`tiles.md`](./tiles.md) — 환경 타일 (바닥/벽/문/락커/책상/계단/출구)
- [`stalkers.md`](./stalkers.md) — 추적자 (idle 2 + walk 4 + detect 1)
- [`props.md`](./props.md) — 소품 (손전등/열쇠/메모/라디오/표지판)
- [`player.md`](./player.md) — 무기 없는 평범한 플레이어 (4방향 × idle 2 + walk 4)
- [`ui.md`](./ui.md) — 패널 프레임, 게이지, 메시지 로그 텍스처

## Claude CoWork 사용 팁

- "이 프롬프트 그대로 SD/ComfyUI 에 넣어서 N장 뽑아줘" — 가능
- "Aseprite 열어서 이 이미지 16×16 그리드로 슬라이스해줘" — 가능
- "spritesheet json (Hash 포맷) 으로 export 해줘" — 가능

각 MD 파일의 **`<!-- prompt -->`** 섹션이 그대로 복사할 부분, **`<!-- direction -->`** 은 사람이 작업 방향 잡을 때 참조용입니다.

## 컨셉 보존을 위한 가이드

- **무기/전투 그림 금지.** 검·총·방패 같은 요소 들어가면 보일러플레이트 컨셉이 깨집니다.
- **추적자 디자인은 모호하게.** 이름표·계급장·뚜렷한 얼굴 X. 실루엣과 분위기로 위협감 전달.
- **풍부한 색감 X.** 12~16색 절제된 팔레트가 호러 톤에 더 잘 맞습니다.
